# AI Face Emotion Detection

This project detects human faces using OpenCV and predicts emotions using DeepFace.

## Technologies Used
- Python
- OpenCV
- DeepFace
- Computer Vision

## Features
- Real-time face detection
- Emotion recognition
- Confidence score display

## Files
face_detection.py – Detects faces using OpenCV  
emotion_detection.py – Detects emotions using DeepFace
